# Alert-go

**Лёгкая и простая библиотека уведомлений для веба**

## Описание

`alert-go` — минималистичная библиотека для создания уведомлений (toast notifications) на веб-сайтах.  
Библиотека проста в установке и использовании, не требует больших зависимостей и позволяет быстро добавить уведомления в проект.

---

## Особенности

- Лёгкая и простая
- Поддержка кастомизации стилей
- Работает в современных браузерах
- Быстрая интеграция в любой фронтенд-проект

---

## Установка

```bash
npm install alert-go
```
